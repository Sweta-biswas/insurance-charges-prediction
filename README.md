# Insurance Charges Prediction

A simple **full-stack machine learning application** that predicts medical insurance charges based on inputs like age, BMI, number of children, smoker status, gender, and region.

The model is trained using **Linear Regression with scikit-learn**, served through a **Flask API**, and connected to a **Next.js + TypeScript frontend** for user interaction.

## Tech Stack
- Python
- scikit-learn
- Flask
- Next.js
- TypeScript
- Tailwind CSS
